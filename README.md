#### 说明
本程序使用Java Swing开发，开发状况如下：<br/>
已实现功能：<br/>
1、象棋棋子走棋规则；<br/>
2、本地UCCI引擎的调用与结果读取；<br/>
3、人机对战；<br/>
4、棋谱记录；<br/>
5、UCCI引擎输出信息日志记录；<br/>
<br/>
未实现功能：<br/>
1、最终输赢判断，<br/>
2、走棋时将军判断；<br/>
3、长将与长拦判断；<br/>
4、菜单和界面按钮的功能；<br/>
<br/><br/>
